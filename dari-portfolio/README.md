# Daria Veselovska — Portfolio

Junior UX/UI Designer portfolio website.

## Structure

```
dari-portfolio/
├── index.html          # Main page
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # JavaScript
├── images/
│   ├── photo.png           # Your photo
│   ├── hero-laptop.png     # Hero section image
│   ├── project-bookstore.png  # Bookstore project card
│   └── project-odesa.png      # Odesa tour project card
├── pages/
│   ├── bookstore.html  # Bookstore project page
│   └── odesa.html      # Odesa tour project page
└── favicon/
    ├── favicon.ico     # Favicon
    └── favicon.svg     # Favicon SVG

```

## How to publish on GitHub Pages

1. Create a new repository on GitHub
2. Upload all files keeping the folder structure
3. Go to Settings → Pages
4. Source: Deploy from a branch → main → / (root)
5. Save — your site will be at `https://yourusername.github.io/repository-name`
